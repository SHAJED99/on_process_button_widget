# Flutter Button Widget

Versatile button widget with customizable styles and animations.

## Features
- Filled, outlined, text, icon styles
- Loading spinner state
- Disabled state
- Custom colors and sizing

## Usage
```dart
OnProcessButton(
  onPressed: () async {
    await someAsyncTask();
  },
  child: Text("Submit"),
)
```

## Installation
```yaml
dependencies:
  on_process_button_widget: ^1.0.0
```
